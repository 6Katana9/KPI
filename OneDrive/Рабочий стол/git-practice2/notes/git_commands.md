git status - показывает текущее состояние репозитория
git add - добавляет изменения в индекс, подготавливая их к созданию коммита
git commit - создает коммит, то есть сохраняет подготовленные изменения в истории
git push - отправляет локальные коммиты в удаленный репозиторий
git pull - получает последние изменения из удаленного репозитория и сразу объединяет их с текущей локальной веткой
git branch - показывает список веток и создает новые ветки
git checkout - переключается между ветками или восстанавливает файлы
