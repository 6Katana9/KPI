- git status - показывает текущее состояние репозитория
- git add - добавляет изменения в индекс
- git commit - сохраняет подготовленные изменения в истории локального репозитория
- git push - отправляет локальные изменения в удаленный репозиторий
- git pull - получает данные из удаленного репозитория и сразу объединяет их с текущей локальной веткой
- git branch - показывает список веток
- git checkout - переключается между веткам
