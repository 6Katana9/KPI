# git status
Показывает текущее состояние репозитория.

# git add
Добавляет изменения в индекс для последующего коммита.

# git commit
Создает снимок изменений в истории проекта.

# git push
Отправляет локальные изменения на удаленный репозиторий.

# git pull
Получает и объединяет изменения из удаленного репозитория.

# git branch
Показывает список веток или создает новую ветку.

# git checkout
Переключает между ветками или версиями файлов.
