Опишите назначение следующих команд:

- git status
показывает изменения и текущий статус репозитория
- git add
добавляет файлы изменения перед коммитом
- git commit
сохраняет все сделанные изменения в локальном репозитории и можно написать комментарий в коммите
- git push
отправляет коммиты в удаленный репозиторий
- git pull
стягивает с удаленного репозитория в локальный репозиторий
- git branch
создает новые ветки
показывает ветки в репозитории
- git checkout
для переключения между ветками
