git status - команда показывает текущее состояние проекта относительно GIT
git add - команда, которая добавляет изменения перед коммитом
git commit - команда фиксирует, комментирует 
изменение и отправляет их из индекса в локальный репозиторий
git push - команда отправляет изменения из локального репозитория в удаленный
git pull - команда стягивает изменения из удаленного репозитория в локальный репозиторий
git branch - команда просматривает нынешнюю ветку/ветки
git checkout - команда позволяет переходить на нужную ветку, которую нужно указать вместе с этой командой
