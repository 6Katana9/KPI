- git status - показывает информацию о нынешнем гите, какая ветка, какие папки/файлы
- git add - добавляет какой-то или все файлы в индексы
- git commit - сохраняет добавленный индекс с добавлением комментария
- git push - отправляет локальный индекс в удаленный репозиторий 
- git pull - стягивает файлы с удаленного репозитория в локальный
- git branch - показывает какие ветки на данный момент есть, git branch 'название ветки' - создает ветку
- git checkout - переходит с одной ветки на выбранную
