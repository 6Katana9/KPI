git status - показывает состояние репозитория на текущий момент: какие файлы изменены, подготовлены к коммиту и какие ещё не отслеживаются

git add - добавляет изменения, подготавливая файлы к сохранению в коммите, папки или файлы попадпают в промежуточную память - индекс

git commit - сохраняет изменения в локальной истории репозитория с комментарием по изменению

git push - отправляеят изменения из локального репозитория в удаленный 
через связку origin

git pull - стягивает изменения из удаленного репозитория по связке origin в локальный репозиторий

git branch - просматривает все ветки, создает новые или удаляет существующие

git checkout - переключает на указанную ветку, между существующими
