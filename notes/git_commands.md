# Git Commands

## git status
Показывает состояние файлов.

## git add
Добавляет изменения в индекс.

## git commit
Создает коммит.

## git push
Отправляет изменения на GitHub.

## git pull
Получает изменения из удаленного репозитория.

## git branch
Показывает список веток.

## git checkout
Переключается между ветками или создает новую.
